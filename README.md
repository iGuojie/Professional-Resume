# Professional Resume Builder

基于 React + TypeScript + Vite 的在线简历生成项目。

## 环境要求

- Node.js >= 18
- npm >= 9

## 本地运行

```bash
npm install
npm run dev
```

默认启动地址：

```text
http://localhost:5173
```

## 构建与预览

```bash
npm run build
npm run preview
```

## 常用脚本

- `npm run dev`: 启动开发服务器
- `npm run build`: 生成生产构建（输出到 `dist/`）
- `npm run preview`: 本地预览生产构建

## 项目结构（简要）

- `App.tsx`: 主页面逻辑
- `components/`: 组件目录
- `docs/`: 项目文档
- `types.ts`: 类型定义
